## üìÇ Project Structure

This project is a static HTML/CSS website containing the following archives:

- **index.html**: Site Index & Brand Story
- **menu.html**: Daily Catalog Menu
- **reading_room.html**: The Roastery & Library Index
- **activities.html**: Community Log & Events
- **location.html**: Location Archive & Access Rights
- **reservation.html**: File Inquiry & Seat Reservation

## üé® Design System

The site follows a strict "Archival Vintage" design system defined in `DESIGN.md`. 
- **Typography**: Courier Prime (Mono) & Space Grotesk (Display)
- **Palette**: Paper Cream (#f4f0e8), Primary Orange (#cf6317), Ink Black (#2d2a26)
- **Mood**: Tactile, bureaucratic, warm, specific.

## üöÄ How to Run Locally

You can run this site with any static file server.

```bash

```

### Using Node.js (Serve)
```bash
npx serve .
```
